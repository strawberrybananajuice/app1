This is a Next.js youtuber helper app that is not finished.
It downloads a YouTube video and parses its srt file. 
It has a C# code for downloading YouTube videos. "dotnet build" might be necessary.

To run,
1. npm install
2. npm run dev
3. Open http://localhost:3001

Once it is running,
1. Go to the Youtube Download Tab.
1-1. Enter the destination folder. 
1-2. Enter YouTube video URL. 
1-3. Press Download Button.
2. It will automatically go to Eng Subtitle tab when downloading is done.
3. Press the green "Process" Buttton and it use ChatGPT to parse the english srt into sentences and also translate them into korean sentences.
