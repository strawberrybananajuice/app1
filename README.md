This is a Next.js youtuber helper app that is not finished.
It downloads a YouTube video and parses its srt file. 
It has a C# code for downloading YouTube videos. "dotnet build" might be necessary.

To run,
1. npm install ("dotnet build" also might be necessary.)
2. npm run dev
3. Open http://localhost:3001

Once it is running,
1. Go to the Youtube Download Tab.
Enter the destination folder. 
Enter YouTube video URL. 
Press Download Button.
2. It will automatically go to Eng Subtitle tab.
3. Press the green "Process" Buttton and it will parse the english srt into sentences and also translate them into korean sentences.
